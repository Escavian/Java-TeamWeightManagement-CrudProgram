# Team Weight Management System (Java)

A menu‑driven Java application that allows the user to enter a **team name**, input **three players**, record each player’s **weight in pounds (lbs)**, and display the team’s total weight in both **lbs** and **kg**. The program also checks whether the team is under the maximum allowed weight limit of **600 lbs**.

## 📌 Features
- Prompt for team name
- Input three players with decimal weight values (e.g., 155.6 lbs)
- Converts total team weight from **lbs → kg**
- Displays:
  - Team name  
  - Player names  
  - Player weights  
  - Total weight in lbs and kg  
  - Team status (Under or Over the 600 lb limit)
- Input validation for decimal weight values
- Clean, simple console interface

## 🧠 Concepts Demonstrated
- CRUD-style data entry (Create + Display)
- Java classes and objects
- Arrays or ArrayLists for storing players
- Decimal input handling using `double`
- Mathematical conversions (lbs → kg)
- Conditional logic for weight limit checks
- Console formatting for clean output
